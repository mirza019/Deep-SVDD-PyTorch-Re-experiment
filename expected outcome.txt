Objective:
----------------------------------------------------------------
Extend the Deep SVDD framework by integrating reconstruction-based learning,
adaptive thresholding, and explainable visualization to improve anomaly detection
accuracy and interpretability while keeping the model lightweight for CPU execution
(MacBook Air M3).

----------------------------------------------------------------
Planned Experiments:
----------------------------------------------------------------

1. Baseline Validation
   Goal: Reproduce Deep SVDD results on MNIST to establish baseline AUC and loss.
   Metrics: AUC-ROC, F1 score, reconstruction error.
   Expected Outcome: Verify implementation and dataset access.

2. Hybrid Deep SVDD + Autoencoder
   Goal: Combine Deep SVDD compactness loss with autoencoder reconstruction loss
         to form a hybrid objective.

Expected Contribution:
----------------------------------------------------------------
This project extends classical Deep SVDD into a "Hybrid Explainable One-Class Framework,"
enhancing accuracy, robustness, and interpretability. It aims to show measurable
performance gains and provide transparent, explainable anomaly visualizations in a
lightweight, reproducible setup.
